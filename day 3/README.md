# 📘 AWS VPC & EC2 Practice – Apache Web Server Setup

## 📖 About This Practice

As part of my AWS learning journey, I created a custom **Virtual Private Cloud (VPC)** and launched an **EC2 instance** inside it. On the EC2 instance, I installed and configured an **Apache Web Server** to host a simple web page.

This exercise helped me understand AWS networking basics and how web servers are deployed in the cloud.

---

## 🧠 What I Practiced

- Creating a custom VPC
    
- Configuring a Public Subnet
    
- Attaching an Internet Gateway
    
- Setting up Route Tables
    
- Launching an EC2 instance
    
- Configuring Security Groups (SSH & HTTP)
    
- Installing and starting Apache Web Server
    
- Deploying a simple HTML webpage
    
- Accessing the application via Public IP
    

---

## 🛠 Technologies Used

- Amazon VPC
    
- Amazon EC2
    
- Apache HTTP Server
    
- Linux (Ubuntu/Amazon Linux)
    
- Basic HTML
    

---

## 🎯 Learning Objective

The purpose of this hands-on practice was to:

- Understand AWS cloud networking
    
- Learn how EC2 instances communicate with the internet
    
- Deploy and configure a web server manually
    
- Gain practical experience with infrastructure setup

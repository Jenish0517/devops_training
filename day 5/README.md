# 📘 Git & GitHub Learning – CLI Practice

## 📖 About This Practice

Today, I learned the fundamentals of **Git** and **GitHub**, including how version control works and how to manage repositories using the command line interface (CLI).

I practiced creating repositories, tracking changes, committing updates, and pushing code to GitHub using Git commands.

---

## 🧠 What I Learned

- What is Version Control
    
- Difference between Git and GitHub
    
- Initializing a local repository
    
- Staging and committing changes
    
- Connecting local repo to GitHub
    
- Pushing code to remote repository
    
- Cloning repositories
    
- Checking repository status
    

---

## 💻 Git Commands Practiced

git init  
git status  
git add .  
git commit -m "Initial commit"  
git branch  
git checkout -b branch-name  
git remote add origin <repo-url>  
git push -u origin main  
git pull  
git clone <repo-url>

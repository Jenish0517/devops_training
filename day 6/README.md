# 📘 Git Commands & Branching – Learning Summary

## 📖 About This Learning

Today, I learned the basics of Git commands and understood how branching works in version control. I practiced managing changes in a local repository and synchronizing them with a remote repository using GitHub.

---

## 🧠 Commands I Learned

### 🔹 `git add`

Stages changes from the working directory to the staging area before committing.

### 🔹 `git commit`

Saves staged changes with a meaningful message to the local repository.

### 🔹 `git push`

Uploads committed changes from the local repository to the remote repository (GitHub).

### 🔹 `git pull`

Fetches and merges the latest changes from the remote repository to the local repository.

### 🔹 `git stash`

Temporarily saves uncommitted changes without committing them.

### 🔹 `git branch`

Creates, lists, or deletes branches.

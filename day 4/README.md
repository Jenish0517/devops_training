# 📘 AWS Hands-On Learning – S3, EC2 & AWS CLI

## 📖 About This Practice

As part of my AWS learning, I explored core AWS services including **Amazon S3**, **Amazon EC2**, and the **AWS CLI**. I practiced hosting a static website using S3 and learned how to manage EC2 and S3 resources through the command line.

This hands-on exercise helped me understand both the AWS Management Console and CLI-based resource management.

---

## 🧠 What I Learned & Practiced

### 🌐 Static Website Hosting using S3

- Created an S3 bucket
    
- Enabled static website hosting
    
- Uploaded HTML/CSS files
    
- Configured bucket policy for public access
    
- Accessed the website using the S3 website endpoint
    

### 💻 Working with EC2

- Launched an EC2 instance
    
- Connected to EC2 using SSH
    
- Understood key pairs and security groups
    

### 🖥 Using AWS CLI

- Installed and configured AWS CLI
    
- Configured credentials using `aws configure`
    
- Managed S3 buckets using CLI commands
    
- Uploaded/downloaded files using CLI
    
- Verified EC2 instance details using CLI
